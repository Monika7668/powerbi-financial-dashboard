# 📊 Financial Performance Dashboard

## 📌 Business Problem

Finance teams rely on manual reporting, leading to delays and lack of real-time insights into revenue and expenses.

---

## 🎯 Objective

Develop an interactive Power BI dashboard to track financial KPIs and improve decision-making.

---

## 🛠 Tech Stack

* Power BI
* SQL
* DAX
* Power Query

---

## 🧹 Data Preparation

* Cleaned and transformed financial data
* Built star schema model
* Created calculated measures using DAX

---

## 📊 Dashboard Features

* Revenue vs Expense tracking
* Profit margin analysis
* Year-over-Year growth
* Region-wise performance
* Interactive filters

---

## 📊 Key Insights

* North region showed consistent growth
* Expenses increased in Q2
* Profit margins improved over time

## 📈 Business Impact

* ⏱ Reduced reporting effort by 3 days

* 📉 Improved financial transparency

## ▶️ How to Run

1. Open Power BI
2. Load dataset from data/ folder
3. Build visuals using fields

## ⚡ Challenges & Solutions

## 🚀 Conclusion

This project enables real-time financial monitoring and supports strategic decisions.
